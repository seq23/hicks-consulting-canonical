Use this page to prepare, review, publish, or hide Hicks Consulting free downloads and premium Gumroad downloads.

How to use this page

  1. Step 1: Find or start a product. Use Smart Filters to find an existing download, then click Edit. To create a new product, skip the table and fill out the form.
  2. Step 2: Add the product details. Choose Free or Premium, add the title, description, category, price, Gumroad link if premium, PDF, and optional cover image.
  3. Step 3: Save, then publish. Save first. Publish only when ready for the product to appear on the public download page. Revoke hides it again.

Status key

  • draft: not public
  • ready_for_approval: complete enough for final review
  • published: visible on public download pages
  • revoked: removed from public download pages
